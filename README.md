# Sooon 統合開発リポジトリ（Submodule構成）

複数のプロジェクトを Git Submodule で管理する親リポジトリです。

## プロジェクト構成

### 1. sooon-sales-os
Sooon 社内CRM＆営業管理プラットフォーム（Next.js, Supabase）

### 2. silas-saas
Google Maps B2B スクレイピング SaaS（Python, Playwright）

### 3. mens-esthe-seo-tools
美容ポータルサイト SEO自動更新ツール

### 4. rokuon
コール録音・文字起こしアプリ

## 使い方

\`\`\`bash
git clone --recurse-submodules git@github.com:atmgmj7-lab/dev-all-projects.git
\`\`\`
